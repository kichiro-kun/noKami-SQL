SingleConnectionManager

UMLClass
noKami-SQL view :: Python Support :: API Architecture :: DBMS Interaction :: Single Manager Component :: SingleConnectionManager

Description

Абстракция описывающая менеджера, то есть сущность которая управляет одичным соединением через соответветствующий интерфейс.

Specialized Elements

  • NoSingleConnectionManager
  • MySQLAdapter
  • PostgreSQLAdapter

Attributes

Visibility Name Type Description
private perform_adapter SingleConnectionInterface

Адаптер соединения, закреплённый к менеджеру. Реализован в стиле паттерна СТРАТЕГИЯ, позволяя динамично менять адаптер при необходимости.

Operations

Visibility Name Description
public set_new_adapter(new_adapter: SingleConnectionInterface)

Закрывает текущее соединение и устанавливает новый адаптер. (соединение не устанавливается, во избежение преждевременной инициализации)
public initialize_new_connection(conn_conf: Dict[str, Any])

Закрывает текущее соединение для инициализации нового.
public reinitialize_active_connection(conn_conf: Dict[str, Any])

Производит переподключение соединения.
public get_active_connection(): SingleConnectionInterface

Возвращает текущее активное соединение, рассчитанное на использование реализованного интерфейса SingleConnectionInterface. Будет использоваться для запросов к БД.
private deconstruct_active_connection()

Приватный метод для уничтожения (закрытия) активного соединения.
public read_active_connection_status(): bool

Возвращает статус активного соединения. (является ли оно активным, то есть подключённым и рабочим для взаимодействия с БД)
